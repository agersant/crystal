# crystal.log.info

Prints a message to the log at the [info](verbosity) level.

## Usage

```lua
crystal.log.info(message)
```

### Arguments

| Name      | Type     | Description                      |
| :-------- | :------- | :------------------------------- |
| `message` | `string` | The message to print in the log. |

## Examples

```lua
crystal.log.info("Player received 120 damage");
```
