# crystal.log.debug

Prints a message to the log at the [debug](verbosity) level.

## Usage

```lua
crystal.log.debug(message)
```

### Arguments

| Name      | Type     | Description                      |
| :-------- | :------- | -------------------------------- |
| `message` | `string` | The message to print in the log. |

## Examples

```lua
crystal.log.debug("Handling jump input");
```
